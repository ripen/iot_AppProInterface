一、SDK包概述

    AnyChat是一套跨平台的音、视频即时通讯解决方案，支持Windows、Linux、Mobile、
Android、iOS、Web等操作系统平台，提供客户端SDK、服务器端SDK，可与第三方平台紧密
集成。

    AnyChat for Web SDK是基于AnyChat for Windows SDK进行再次封装的插件，提供纯
JavaScript编程接口，支持Windows操作系统上的主流浏览器，包括IE、Firefox、Chrome等。

    AnyChat for Web SDK可与其它平台的AnyChat互联互通，连接同一个服务器程序，有关
服务器的SDK可参考AnyChat Platform Core SDK开发包。

    AnyChat for Web SDK包里面提供了Demo程序源代码供参考，安装插件后，直接有浏览器
打开即可运行。
